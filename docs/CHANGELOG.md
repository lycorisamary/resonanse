## [Неделя 2] - 2024

### Исправления

#### Swagger UI авторизация
- ✅ **Исправлена ошибка 422 в Swagger UI**
  - Заменен ручной парсинг заголовка Authorization на стандартный `HTTPBearer` из FastAPI
  - Теперь Swagger UI корректно работает с JWT токенами
  - Создан файл `app/core/dependencies.py` для централизованного управления зависимостями

**Как использовать:**
1. Откройте http://localhost:8000/docs
2. Войдите через `/auth/login` и скопируйте `access_token`
3. Нажмите кнопку "Authorize" (🔒)
4. Вставьте токен в поле "Value" (без слова "Bearer")
5. Нажмите "Authorize", затем "Close"
6. Все защищенные эндпоинты теперь работают!

### Как протестировать

#### Backend (Swagger)

1. Запустите бэкенд: `uvicorn app.main:app --reload`
2. Откройте http://localhost:8000/docs
3. Зарегистрируйтесь через `/auth/register`
4. Войдите через `/auth/login` и скопируйте токен
5. Нажмите "Authorize" и вставьте токен
6. Протестируйте `/users/me` - должно работать без ошибок!

#### Frontend

1. Настройте `baseUrl` в `lib/config/api_config.dart`
2. Запустите: `flutter run`
3. Зарегистрируйтесь или войдите
4. Перейдите в профиль и загрузите аватар
5. Отредактируйте профиль
